# MyMeds

This is a Progressive Web Application (PWA) built with Vue.js and Vite.

## Project Setup

### Install dependencies

```bash
npm install
```

### Compiles and Hot-Reloads for Development

```bash
npm run dev
```

### Compiles and Minifies for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## PWA Features

This project uses `vite-plugin-pwa` for PWA capabilities.
The configuration is located in `vite.config.js`.

Note: You will need to add `pwa-192x192.png` and `pwa-512x512.png` to the `public` directory for the PWA manifest to be fully valid.
